# 📊 Customer Purchase Behavior Analysis Dashboard

## 🚀 Project Overview
This project analyzes e-commerce data to identify customer purchasing patterns, revenue trends, and high-value customers.

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Streamlit

## 📊 Features
- Top products analysis
- Revenue by country
- Monthly sales trends
- High-value customer identification
- Interactive dashboard with filters

## ▶️ How to Run
```bash
pip install streamlit pandas matplotlib
streamlit run app.py
