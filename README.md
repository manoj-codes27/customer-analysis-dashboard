# 📊 Customer Purchase Behavior Analysis Dashboard

## 🚀 Overview
An interactive dashboard built using Streamlit to analyze customer purchasing behavior and sales trends from e-commerce data.

## 🛠 Tech Stack
- Python
- Pandas
- Matplotlib
- Streamlit

## 📂 Dataset

Download the dataset from:
https://www.kaggle.com/datasets/vijayuv/onlineretail

File used:
**OnlineRetail.csv**

## 📊 Features
- Top-selling products analysis
- Revenue by country
- Monthly sales trends
- High-value customer identification
- Interactive filters
